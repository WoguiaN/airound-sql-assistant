# AiRound AI SQL Assistant
Als Werkstudent Data Analyst bei Hamburgerenergienetz (HNE) ein großer Konzern mit tausenden Daten, mein Team (eRound) sitzt ab und zu vor Personnalmangel und daher ist es empfehlenswert ein kI System zu entwickeln, 
das uns effizient mit den Datenverarbeitung unterstützen wird.
Das System heißt   AiRoud ( also von eRound => AiRound) und wird von mir kontinuirlich entwickelt.

AI-powered SQL assistant built with:
- Streamlit
- Google Gemini
- SQLite

Users can ask questions in natural language and retrieve SQL data automatically.
